# Financify 
A comprehensive Personal Finance Management System built with Python.

##  Features
* Dashboard: Visual breakdown of expenses with Matplotlib pie charts.
* Budgeting: Set monthly limits for specific categories.
* Reports: Compare Income vs. Expenses with bar charts.
* Data Safety:Secure login and local SQLite database storage.
##  How to Run
1. Install dependencies: `pip install -r requirements.txt`
2. Run the app: `python login.py`
